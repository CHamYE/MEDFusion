# Feature-level Multi-Modality Medical Image Fusion
## 📌 Overview
Feature-level fusion is the **most mainstream fusion paradigm** in the current multi-modality medical image fusion (MMIF) field. Its core workflow is: first, extract high-dimensional features of each modality image through feature extractors (traditional transform domain methods, CNN, Transformer, GAN, etc.), then complete the integration and optimization of cross-modal information in the feature space, and finally reconstruct the fused image.

This paradigm can effectively filter redundant information and noise in the original image, accurately capture the specific complementary features of each modality (e.g., anatomical structure features, functional metabolic features), and has strong adaptability to downstream tasks. It can be seamlessly connected to clinical tasks such as medical image segmentation, detection, and diagnosis, and is the core technical direction of current academic research and clinical implementation.
