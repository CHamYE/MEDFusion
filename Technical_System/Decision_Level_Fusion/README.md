# Decision-level Multi-Modality Medical Image Fusion
## 📌 Overview
Decision-level fusion is the high-level fusion paradigm in multi-modality medical image fusion (MMIF). For each modality image, it completes independent analysis, reasoning and decision output (e.g., segmentation mask, diagnostic classification result, lesion localization) first, then integrates and optimizes multiple independent decision results to output the final unified decision.

This paradigm has lower requirements for modality registration accuracy, with extremely strong robustness and anti-interference ability. It is especially suitable for clinical multi-modal joint diagnosis, multi-model ensemble segmentation and other scenarios, and is an important technical path for the clinical implementation of medical imaging AI.
