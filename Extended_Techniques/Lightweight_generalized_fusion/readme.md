# Lightweight & Real-Time Optimization for Multi-Modality Medical Image Fusion
## 📌 Overview
Lightweight and real-time performance are key prerequisites for the clinical deployment of multi-modal medical image fusion (MMIF) technology, especially in edge scenarios such as intraoperative navigation, mobile ultrasound, and primary hospital imaging equipment. Most existing high-performance fusion methods have a large number of parameters and high computational complexity, which are difficult to deploy on devices with limited computing resources, and cannot meet the real-time requirements of clinical scenarios.

This section summarizes the representative lightweight and real-time optimization methods for MMIF, which mainly realize lightweight design through depthwise separable convolution, neural architecture search (NAS), low-rank adaptation, model distillation, and other technical paths.

## 📑 Curated List of Methods
| Full Paper Title | Publication Venue & Year | Core Contributions | Paper Link | Open-Source Code |
|-------------------|---------------------------|--------------------|------------|------------------|
| LRFNet: a real-time medical image fusion method guided by detail information | Computers in Biology and Medicine (CBM), 2024 | Proposes LRFNet, a real-time lightweight medical image fusion network guided by detail information. It designs a lightweight residual unit (LRU) based on depthwise convolution and channel shuffle, with only 253kB parameters (6.7% of EMFusion). The inference speed is 0.005s per frame, which is 33% faster than SDNet, and can be deployed on edge devices. | [Link](https://www.sciencedirect.com/science/article/pii/S0010482524004657) | Not publicly available |
| Learning to search a lightweight generalized network for medical image fusion | IEEE Transactions on Circuits and Systems for Video Technology (TCSVT), 2023 | Proposes ALMF-Net, a lightweight generalized network for medical image fusion via hardware-aware NAS. It embeds hardware constraints (FLOPs<1G, parameters<2M) into the search space, and the searched ALMF-Net has only 1.4M parameters (72% less than U2Fusion), with 0.08s per frame inference speed, meeting the needs of clinical edge deployment. | [Link](https://ieeexplore.ieee.org/abstract/document/10360160/) | [GitHub](https://github.com/RollingPlain/ALMFnet) |
| SDNet: A Versatile Squeeze-and-Decomposition Network for Real-Time Image Fusion | International Journal of Computer Vision (IJCV), 2021 | Proposes SDNet, a versatile squeeze-and-decomposition network for real-time image fusion including medical MMIF. It has only 0.287M parameters, and the inference speed of 256×256 images is 0.004s per frame on GPU, which is 51 times faster than U2Fusion and 80 times faster than DenseFuse, supporting real-time clinical scenarios. | [Link](https://link.springer.com/article/10.1007/s11263-021-01501-8) | [GitHub](https://github.com/HaoZhang1018/SDNet) |
| Zero-Learning Fast Medical Image Fusion | 22th International Conference on Information Fusion (FUSION), 2019 | Proposes a zero-learning fast medical image fusion method, which uses a pre-trained VGG-16 network without task-specific training. The inference speed is 0.006s per frame on GPU and 0.03s per frame on CPU, which can meet the real-time requirements of clinical scenarios, and supports any number of modalities. | [Link](https://ieeexplore.ieee.org/document/9011178) | [GitHub](https://github.com/IVRL/Zero-Learning-Medical-Fusion) |
